# DocsAppointment-

Frontend  ->  React.js , Context API ,Tailwind Css  
Backend   ->  Node.js , Express    
Database  ->  MongoDb    
ODM       ->  Mongoose

To run locally ->

1)  Download
2)  cd appointment
3)  npm run server   //(backend server start on localhost 5000)
4)  cd frontend
5)  npm start       //(frontend start on localhost 3000)
